# User Journey & App Flow

## 1. Authentication Flow
- **Landing Page**: Value proposition and "Get Started" / "Login" buttons.
- **Register**: Onboarding form (Name, Email, DOB, Last Period Start, Cycle Length).
- **Login**: Secure access to existing account.
- **Forgot Password**: Email-based recovery flow.

## 2. Main Navigation (Bottom Tabs)
- **Dashboard (Home)**: 
  - Current cycle status and countdown.
  - Quick action buttons (Log Mood, Chat with HIM).
  - Wellness score and streak stats.
- **Tracker (Calendar)**:
  - Visual monthly view of phases and symptoms.
  - "Log Period" modal for updating flow dates.
  - Link to "Detailed Insights".
- **Chat (Assistant)**:
  - Real-time messaging with HIM AI.
  - Voice call mode for hands-free conversation.
  - Message history persistence.
- **Wellness (Hub)**:
  - Affirmation of the day.
  - Tips for the current phase.
  - Category-based library (Nutrition, Fitness, etc.).
  - Link to "Audio/Media Library".
- **Profile (Settings)**:
  - Personal info management.
  - Theme selector (Blossom, Pookie, Dark Nite, etc.).
  - Language selector (Hindi, Bengali, etc.).
  - Partner Mode toggle.

## 3. Sub-Screens & Interactions
- **Insights**: Detailed charts for mood trends, symptom distribution, and cycle history.
- **Games**: Points dashboard, badge collection, and arcade mini-games.
- **Log Period**: Step-by-step form for logging flow intensity and symptoms.
- **Partner Mode**: Invite partner via email to share phase-specific support tips.
- **Audio Library**: Embedded YouTube player for guided meditations and lofi music.

## 4. Interaction Rationale
- **Single Tap Log**: Every core action is reachable within 2 taps from the dashboard.
- **Emotional Palette**: Transitioning between screens uses subtle fades and glassmorphic transitions to maintain a calming atmosphere.
